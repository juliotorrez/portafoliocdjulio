01
¿Qué es HTML5?
Fundamentos y origen del lenguaje de la web

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. No es un lenguaje de programación como tal: es un sistema de etiquetas que le dice al navegador qué es cada elemento de una página (un título, un párrafo, una imagen, un botón...).

La versión HTML5, publicada oficialmente en 2014 por el W3C, es la versión actual y trae mejoras enormes: soporte nativo para video y audio, etiquetas semánticas, Canvas para gráficos, APIs de geolocalización, almacenamiento local y mucho más.

Fue inventado por Tim Berners-Lee en 1991 (¡el mismo que conocimos en la Tarea 01!). Desde entonces, HTML ha evolucionado hasta convertirse en el pilar de toda la web moderna.

💡
¿Cómo funciona? Tú escribes el código HTML, el navegador (Chrome, Firefox, Safari...) lo lee y lo convierte en la página visual que ves. El navegador es el intérprete de HTML.
🌐
Tecnologías hermanas: HTML define la estructura, CSS define el estilo visual, y JavaScript define el comportamiento interactivo. Los tres juntos forman el desarrollo web front-end.

¿Cómo luce internamente un documento HTML? Todo documento tiene una jerarquía de elementos anidados, como un árbol:

<!DOCTYPE html> ← declara que es HTML5
<html lang="es"> ← elemento raíz
<head> ← metadatos (invisible)
<meta charset="UTF-8">
<title>Mi página web</title>
</head>
<body> ← contenido visible
<h1>Título principal</h1>
<p>Un párrafo de texto</p>
</body>
</html>
02
Estructura Básica
Etiquetas fundamentales de todo documento HTML
mi-primera-pagina.html ESTRUCTURA 
<!-- 1. Declaración DOCTYPE (siempre va al inicio) -->
<!DOCTYPE html>

<!-- 2. Elemento raíz con idioma -->
<html lang="es">

  <!-- 3. Cabecera (metadatos, no visible) -->
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Mi primera página web">
    <title>Mi Primera Página</title>
    <link rel="stylesheet" href="estilos.css">
  </head>

  <!-- 4. Cuerpo (todo lo visible) -->
  <body>
    <h1>¡Hola desde HTML5!</h1>
    <p>Esta es mi primera página web.</p>
    <script src="app.js"></script>
  </body>

</html>
EtiquetaFunciónTipo
<!DOCTYPE html>Declara la versión de HTML (HTML5). Va siempre en la primera línea.meta
<html>Elemento raíz que envuelve todo el documento.block
<head>Contiene metadatos: título, charset, enlaces a CSS, scripts, SEO.meta
<meta>Define metadatos como charset (codificación), viewport (responsive), descripción.meta
<title>Define el título que aparece en la pestaña del navegador.meta
<link>Vincula recursos externos como hojas de estilo CSS.meta
<body>Contiene todo el contenido visible de la página.block
<script>Incluye o vincula código JavaScript.block
<style>Escribe CSS directamente dentro del HTML (no recomendado para proyectos grandes).meta
⚠️
Regla importante: Las etiquetas HTML se abren <etiqueta> y se cierran </etiqueta>. Algunas etiquetas son "auto-cerradas" y no necesitan etiqueta de cierre: <meta>, <link>, <br>, <img>, <input>.
03
Etiquetas de Texto
Encabezados, párrafos, énfasis y formato
texto.html TEXTO 
<!-- Encabezados: del más importante (h1) al menos (h6) -->
<h1>Título Principal (solo 1 por página)</h1>
<h2>Subtítulo de sección</h2>
<h3>Sub-sección</h3>
<h4>Nivel 4</h4>

<!-- Párrafo -->
<p>Esto es un párrafo de texto normal.</p>

<!-- Énfasis y formato -->
<p>
  Texto <strong>muy importante</strong> (negrita semántica).
  Texto <em>con énfasis</em> (cursiva semántica).
  Texto <mark>resaltado</mark> en amarillo.
  Texto <del>tachado</del> y <ins>insertado</ins>.
  Fórmula: H<sub>2</sub>O y E=mc<sup>2</sup>.
</p>

<!-- Salto de línea y línea horizontal -->
<br>  <!-- salto de línea, NO usa cierre -->
<hr>  <!-- línea horizontal divisoria -->

<!-- Citas -->
<blockquote>
  La Web es para todos. — Tim Berners-Lee
</blockquote>

<!-- Código y preformateado -->
<code>console.log("Hola")</code>
<pre>Texto
  con espacios
    respetados</pre>

<!-- Dirección y tiempo -->
<address>Tuxtla Gutiérrez, Chiapas</address>
<time datetime="2025-01-15">15 de enero de 2025</time>
EtiquetaDescripciónTipo
<h1> … <h6>Encabezados. h1 es el más importante (usado para el título principal de la página).block
<p>Párrafo de texto. El navegador agrega margen arriba y abajo automáticamente.block
<strong>Texto importante/negrita con valor semántico (SEO lo considera relevante).inline
<em>Texto con énfasis/cursiva semántica.inline
<mark>Texto resaltado (como con marcador).inline
<del> / <ins>Texto eliminado / insertado (tachado o subrayado).inline
<sub> / <sup>Subíndice (H₂O) y superíndice (m²).inline
<br>Salto de línea. Es auto-cerrada, no tiene contenido.inline
<hr>Línea horizontal divisoria temática.block
<blockquote>Cita larga de otra fuente. Se muestra con sangría.block
<q>Cita breve en línea. El navegador agrega comillas automáticas.inline
<code>Fragmento de código en línea (fuente monoespaciada).inline
<pre>Texto preformateado: respeta espacios y saltos de línea tal cual.block
<span>Contenedor inline genérico para aplicar estilos o JS a parte de un texto.inline
<div>Contenedor block genérico para agrupar elementos y aplicar CSS.block

Listas y Tablas
Organización de información estructurada
listas.html LISTAS 
<!-- Lista desordenada (viñetas) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Lista ordenada (numerada) -->
<ol>
  <li>Aprender HTML5</li>
  <li>Aprender CSS3</li>
  <li>Aprender JavaScript</li>
</ol>

<!-- Lista de definición -->
<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para la web</dd>
  <dt>CSS</dt>
  <dd>Hojas de estilo en cascada</dd>
</dl>
tabla.html TABLAS 
<table>
  <!-- Cabecera de la tabla -->
  <thead>
    <tr>  <!-- tr = table row (fila) -->
      <th>Lenguaje</th>  <!-- th = table header -->
      <th>Uso</th>
      <th>Dificultad</th>
    </tr>
  </thead>
  <!-- Cuerpo de la tabla -->
  <tbody>
    <tr>
      <td>HTML</td>  <!-- td = table data (celda) -->
      <td>Estructura web</td>
      <td>Fácil</td>
    </tr>
    <tr>
      <td colspan="2">Celda que ocupa 2 columnas</td>
      <td>Media</td>
    </tr>
  </tbody>
  <tfoot>
    <tr><td colspan="3">Pie de tabla</td></tr>
  </tfoot>
</table>
💡
colspan y rowspan: colspan="n" hace que una celda ocupe n columnas. rowspan="n" hace que ocupe n filas. Son muy útiles para tablas complejas.

Multimedia y Enlaces
Imágenes, video, audio y navegación entre páginas
multimedia.html MEDIA 
<!-- ===== ENLACES ===== -->
<!-- Enlace externo (abre en nueva pestaña) -->
<a href="https://www.google.com" target="_blank" rel="noopener">
  Ir a Google
</a>

<!-- Enlace interno (dentro del sitio) -->
<a href="tarea-01.html">Ver Tarea 01</a>

<!-- Ancla interna (ir a sección de la misma página) -->
<a href="#modulo3">Ir al módulo 3</a>

<!-- Enlace de correo y teléfono -->
<a href="mailto:correo@ejemplo.com">Enviar correo</a>
<a href="tel:+529611234567">Llamar</a>

<!-- ===== IMÁGENES ===== -->
<img
  src="foto.jpg"
  alt="Descripción de la imagen (obligatorio)"
  width="400"
  height="300"
  loading="lazy"
>

<!-- Imagen con figura y pie de foto -->
<figure>
  <img src="nutria.jpg" alt="Una nutria con laptop">
  <figcaption>La nutria programadora 🦦</figcaption>
</figure>

<!-- ===== VIDEO ===== -->
<video controls width="640" poster="miniatura.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta video.
</video>

<!-- ===== AUDIO ===== -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<!-- ===== IFRAME (incrustar otra página) ===== -->
<iframe
  src="https://www.youtube.com/embed/dQw4w9WgXcQ"
  width="560" height="315"
  allowfullscreen
></iframe>
EtiquetaDescripciónTipo
<a>Ancla/enlace. El atributo href define el destino.inline
<img>Imagen. Requiere src (ruta) y alt (texto alternativo). Auto-cerrada.media
<figure>Contenedor semántico para contenido ilustrativo (imagen, gráfica, código).block
<figcaption>Pie de foto o descripción de un <figure>.block
<video>Reproduce video HTML5 nativo. Atributos: controls, autoplay, loop, muted.media
<audio>Reproduce audio HTML5 nativo.media
<source>Define fuentes alternativas para video o audio. Auto-cerrada.media
<iframe>Incrusta otra página o contenido externo (videos de YouTube, mapas, etc.).block
<canvas>Lienzo para dibujo con JavaScript (gráficos, animaciones, juegos).media
<svg>Gráficos vectoriales escalables directamente en HTML.media

Formularios
Recibir datos del usuario: inputs, botones y validación
formulario.html FORMS 
<form action="/enviar" method="POST">

  <!-- Texto simple -->
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre"
         placeholder="Tu nombre" required>

  <!-- Email con validación automática -->
  <input type="email" name="correo" placeholder="tu@correo.com">

  <!-- Contraseña -->
  <input type="password" name="pass" minlength="8">

  <!-- Número con rango -->
  <input type="number" name="edad" min="1" max="120">

  <!-- Fecha -->
  <input type="date" name="nacimiento">

  <!-- Color -->
  <input type="color" name="favorito" value="#7ab8e0">

  <!-- Checkbox y Radio -->
  <input type="checkbox" name="html" id="html" checked>
  <label for="html">Me gusta HTML</label>

  <input type="radio" name="nivel" value="principiante">
  <label>Principiante</label>

  <!-- Select / desplegable -->
  <select name="lenguaje">
    <option value="">Elige un lenguaje</option>
    <option value="html">HTML</option>
    <option value="css">CSS</option>
  </select>

  <!-- Área de texto -->
  <textarea name="mensaje" rows="4" cols="40"
             placeholder="Escribe tu mensaje..."></textarea>

  <!-- Botones -->
  <button type="submit">Enviar formulario</button>
  <button type="reset">Limpiar</button>
  <input type="file" accept=".jpg,.png">

</form>
✅
Validación nativa HTML5: Los atributos required, min, max, minlength, maxlength, pattern validan los campos automáticamente sin JavaScript. El tipo email valida el formato del correo; url valida URLs.
07
HTML Semántico
Etiquetas con significado: accesibilidad y SEO

El HTML semántico usa etiquetas que describen el significado del contenido, no solo su apariencia. Esto es crucial para: el SEO (los motores de búsqueda entienden mejor tu página), la accesibilidad (los lectores de pantalla para personas con discapacidad visual funcionan mejor) y la mantenibilidad del código.

Antes de HTML5 todo se hacía con <div>. Ahora tenemos etiquetas que describen exactamente qué es cada parte de la página.

estructura-semantica.html SEMÁNTICA 
<body>

  <!-- Encabezado del sitio (logo, menú principal) -->
  <header>
    <nav>
      <ul>
        <li><a href="/">Inicio</a></li>
        <li><a href="/acerca">Acerca</a></li>
      </ul>
    </nav>
  </header>

  <!-- Contenido principal (único por página) -->
  <main>

    <!-- Sección temática con encabezado -->
    <section>
      <h2>Sobre HTML5</h2>
      <p>Contenido de la sección...</p>
    </section>

    <!-- Artículo independiente (post, noticia) -->
    <article>
      <header>
        <h2>Historia de la Web</h2>
        <time datetime="1991-08-06">6 de agosto de 1991</time>
      </header>
      <p>Tim Berners-Lee publica la primera web...</p>
      <footer>Autor: Tim B-L</footer>
    </article>

    <!-- Contenido relacionado pero separable -->
    <aside>
      <h3>¿Sabías que...?</h3>
      <p>HTML tiene más de 100 etiquetas.</p>
    </aside>

    <!-- Detalles expandibles -->
    <details>
      <summary>Ver más información</summary>
      <p>Contenido oculto que se expande al hacer clic.</p>
    </details>

  </main>

  <!-- Pie de página del sitio -->
  <footer>
    <p>&copy; 2025 Mi Portafolio Digital</p>
  </footer>

</body>
EtiquetaDescripción semánticaTipo
<header>Encabezado de la página o de una sección. Suele contener logo y navegación.block
<nav>Bloque de navegación principal (menú de links).block
<main>Contenido principal único de la página. Solo debe haber uno.block
<section>Sección temática con su propio encabezado. Agrupa contenido relacionado.block
<article>Contenido autónomo y reutilizable: post de blog, noticia, tarjeta.block
<aside>Contenido secundario relacionado: barra lateral, notas al margen.block
<footer>Pie de página del sitio o de un artículo.block
<details>Widget de contenido colapsable/expandible sin JavaScript.block
<summary>Título visible del <details>.inline
<dialog>Ventana modal/diálogo nativo de HTML5.block
<template>Fragmento HTML inactivo que JavaScript puede clonar e insertar.meta
08
Atributos Globales
Atributos que puedes usar en cualquier etiqueta HTML

Los atributos globales son especiales porque funcionan en cualquier etiqueta HTML, no solo en algunas. Son la clave para conectar HTML con CSS e JavaScript.

🆔
id="nombre"
Identificador único en toda la página. Usado por CSS (#id) y JavaScript (getElementById).
🏷️
class="nombre"
Clase reutilizable. Múltiples elementos pueden tener la misma clase. Usado por CSS (.clase).
🎨
style="..."
CSS en línea directamente en el elemento. Evitar en proyectos grandes; mejor usar clases CSS.
💬
title="texto"
Texto emergente (tooltip) que aparece al pasar el cursor sobre el elemento.
🚫
hidden
Oculta el elemento (equivale a display:none). JavaScript puede quitarlo para mostrarlo.
♿
tabindex="n"
Controla el orden de navegación con Tab. Esencial para accesibilidad del teclado.
atributos-globales.html ATRIBUTOS 
<!-- Atributos de datos personalizados (data-*) -->
<div
  id="tarjeta-1"
  class="tarjeta activa"
  data-lenguaje="html"
  data-nivel="1"
  title="Tarjeta de HTML"
  style="background: #c7eff8;"
>
  Contenido de la tarjeta
</div>

<!-- Atributos de accesibilidad (ARIA) -->
<button
  aria-label="Cerrar menú"
  aria-expanded="false"
  role="button"
>
  ✕
</button>

<!-- Atributos de idioma y dirección -->
<p lang="en">This text is in English.</p>
<p dir="rtl">Texto de derecha a izquierda</p>

<!-- contenteditable: hace el elemento editable -->
<div contenteditable="true">
  Puedes editar este texto directamente.
</div>

<!-- draggable: permite arrastrar el elemento -->
<img src="logo.png" draggable="true" alt="Logo">
📌
Atributos data-*: Puedes crear tus propios atributos con el prefijo data- para guardar información extra en los elementos. JavaScript puede leerlos con element.dataset.lenguaje. Son muy usados en frameworks modernos.

🎓 ¿Qué aprendiste?

Recorriste los 8 módulos fundamentales de HTML5: desde su historia y estructura básica, hasta etiquetas de texto, listas, tablas, multimedia, formularios, semántica avanzada y atributos globales. Con estos conocimientos ya puedes construir páginas web completas y bien estructuradas.

Recuerda que HTML es solo el esqueleto. El siguiente paso es combinarlo con CSS3 para darle estilo visual, y con JavaScript para hacerlo interactivo. Juntos, estos tres lenguajes son el trío esencial del desarrollo web front-end.

Y como lo vimos en la Tarea 01: gracias a Tim Berners-Lee, este lenguaje es libre, abierto y accesible para todos. ¡Ahora te toca a ti crear con él! 🦦
