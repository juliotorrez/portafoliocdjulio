01
¿Qué es CSS3?
Fundamentos, historia y formas de usar CSS

CSS (Cascading Style Sheets — Hojas de Estilo en Cascada) es el lenguaje que controla la apariencia visual de los documentos HTML. Mientras HTML define qué hay en una página, CSS define cómo se ve: colores, tipografía, espaciado, animaciones, layouts y mucho más.

CSS3 es la versión actual, dividida en módulos independientes que se actualizan por separado: selectores, animaciones, Flexbox, Grid, variables personalizadas, filtros, y más. Fue desarrollado por el W3C y es soportado por todos los navegadores modernos.

La palabra "cascading" (en cascada) describe cómo se aplican los estilos: cuando hay reglas que compiten, el navegador sigue reglas de especificidad y herencia para decidir cuál aplica. Entender esto es clave para dominar CSS.

🔗
Las 3 formas de escribir CSS:
1. CSS externo — archivo .css vinculado con <link> en el <head>. La más recomendada.
2. CSS interno — dentro de una etiqueta <style> en el <head>.
3. CSS en línea — dentro del atributo style="" de una etiqueta. Solo para casos puntuales.
sintaxis-basica.css SINTAXIS 
/* ===== Anatomía de una regla CSS ===== */
/* selector { propiedad: valor; } */

h1 {
  color: #2c4a6e;           /* propiedad: valor */
  font-size: 2rem;
  font-family: 'Baloo 2', cursive;
  margin-bottom: 16px;
}

/* Múltiples selectores */
h1, h2, h3 {
  font-weight: 800;
  line-height: 1.2;
}

/* Comentario de una línea */
p {
  font-size: 1rem;          /* relativo al elemento padre */
  line-height: 1.7;        /* espacio entre líneas */
  color: #476a8a;
}
02
Selectores CSS
Cómo apuntar exactamente al elemento que quieres estilizar

Los selectores son patrones que indican a qué elementos HTML se aplicarán los estilos. Son la base de CSS: si no sabes seleccionar bien, no puedes diseñar bien. Existen selectores básicos, de combinación, de pseudo-clase y de pseudo-elemento.

p
Selector de tipo
Selecciona todos los elementos de ese tipo HTML. Aplica a todos los <p>.
.clase
Selector de clase
Selecciona todos los elementos con ese class="". El más usado en proyectos reales.
#id
Selector de ID
Selecciona el único elemento con ese id="". Alta especificidad; usar con moderación.
*
Selector universal
Selecciona absolutamente todos los elementos. Muy usado para reset CSS.
A B
Descendiente
Selecciona B cuando está dentro de A, a cualquier profundidad.
A > B
Hijo directo
Selecciona B solo cuando es hijo inmediato de A (no nietos).
:hover
Pseudo-clase
Estado del elemento. :hover, :focus, :active, :nth-child(n), :first-child, :not().
::before
Pseudo-elemento
Crea elementos virtuales. ::before, ::after, ::placeholder, ::selection, ::first-line.
selectores.css SELECTORES 
/* ===== Selectores básicos ===== */
p                  { color: #476a8a; }    /* tipo */
.tarjeta           { border-radius: 16px; }  /* clase */
#portada           { height: 100vh; }     /* id */
*                  { box-sizing: border-box; }

/* ===== Combinadores ===== */
.nav a             { text-decoration: none; }   /* descendiente */
.lista > li        { margin-bottom: 8px; }    /* hijo directo */
h2 + p             { margin-top: 0; }          /* hermano adyacente */

/* ===== Pseudo-clases ===== */
.boton:hover       { background: #2c6ea6; }
.boton:active      { transform: scale(.97); }
input:focus        { outline: 2px solid #7ab8e0; }
li:first-child     { font-weight: 700; }
li:nth-child(2n)   { background: #f4f9ff; }   /* filas pares */
p:not(.especial)   { font-size: 1rem; }

/* ===== Pseudo-elementos ===== */
.seccion::before   {
  content: "★ ";
  color: #7ab8e0;
}
::selection        { background: #b8d8f0; }  /* texto seleccionado */
::placeholder      { color: #94a3b8; }
⚖️
Especificidad CSS: Cuando dos reglas apuntan al mismo elemento, gana la de mayor especificidad. El orden de poder es: !important > style="" > #id > .clase > etiqueta. Evita usar !important excepto como último recurso.
03
El Modelo de Caja (Box Model)
Todo en CSS es una caja: aprende a controlar su tamaño y espacio

El Box Model es el concepto más importante de CSS. Todo elemento HTML es tratado como una caja rectangular con 4 capas: content (contenido), padding (relleno interior), border (borde) y margin (margen exterior). Dominar esto es entender cómo se posicionan y dimensionan todos los elementos.

margin
border
padding
content
box-model.css BOX MODEL 
/* ===== Box Model completo ===== */
.tarjeta {
  /* CONTENT: tamaño del contenido */
  width:  300px;
  height: 200px;

  /* PADDING: espacio interior (entre contenido y borde) */
  padding: 20px;                    /* los 4 lados */
  padding: 10px 20px;             /* arriba/abajo  izquierda/derecha */
  padding: 10px 20px 14px 8px;  /* arriba derecha abajo izquierda */
  padding-top:    10px;
  padding-right:  20px;
  padding-bottom: 14px;
  padding-left:   8px;

  /* BORDER: borde del elemento */
  border: 2px solid #7ab8e0;
  border-radius: 16px;
  border-top: 4px solid #3a7bbf;

  /* MARGIN: espacio exterior (entre este y otros elementos) */
  margin: 20px;
  margin: 0 auto;              /* centrar horizontalmente */

  /* BOX-SIZING: hace que padding y border no sumen al width */
  box-sizing: border-box;      /* SIEMPRE usar esto */

  /* DISPLAY: tipo de caja */
  display: block;             /* bloque (ocupa toda la línea) */
  display: inline;            /* en línea (solo su contenido) */
  display: inline-block;       /* en línea pero con tamaño */
  display: none;              /* oculto, sin espacio */
}
PropiedadDescripciónTipo
width / heightDimensiones del área de contenido.box
max-width / min-widthLímites de ancho. max-width: 100% es clave para imágenes responsivas.box
paddingEspacio interior entre contenido y borde. Afecta el tamaño si no usas border-box.box
marginEspacio exterior. margin: 0 auto centra elementos block con ancho definido.box
borderBorde del elemento. Shorthand: grosor estilo color. Ej: 2px solid #aaa.box
border-radiusBordes redondeados. 50% crea un círculo perfecto.visual
box-sizingborder-box hace que padding y border estén incluidos en width/height.box
overflowQué pasa cuando el contenido supera el tamaño: visible, hidden, scroll, auto.box
displayTipo de caja: block, inline, inline-block, flex, grid, none.layout
positionPosicionamiento: static, relative, absolute, fixed, sticky.layout
top / right / bottom / leftCoordenadas de posición (solo funcionan con position no-static).layout
z-indexOrden de apilamiento (quién queda encima de quién). Mayor número = encima.layout
box-shadowSombra. Ej: 0 4px 20px rgba(0,0,0,.15). Se pueden apilar varias.visual
opacityTransparencia del elemento (0 = invisible, 1 = opaco).visual
04
Texto y Tipografía
Propiedades para controlar fuentes, tamaño y legibilidad
tipografia.css TEXTO 
/* ===== Importar fuente de Google Fonts ===== */
@import url('https://fonts.googleapis.com/css2?family=Nunito:wght@400;700&display=swap');

body {
  /* Familia tipográfica */
  font-family: 'Nunito', sans-serif;

  /* Tamaño base */
  font-size:   16px;          /* absoluto */
  font-size:   1rem;          /* relativo al root (recomendado) */

  /* Altura de línea */
  line-height: 1.7;

  /* Color */
  color:       #2c4a6e;
}

.titulo {
  font-size:      clamp(1.5rem, 4vw, 3rem);  /* responsivo automático */
  font-weight:    800;
  letter-spacing: -0.5px;
  text-transform: uppercase;
  text-align:     center;
  text-shadow:    2px 2px 0 rgba(0,0,0,.15);
}

.parrafo {
  text-decoration:   underline;     /* subrayado */
  text-decoration:   none;         /* quita subrayado de enlaces */
  text-overflow:     ellipsis;     /* texto largo → "..." */
  white-space:       nowrap;       /* no rompe línea */
  word-break:        break-word;   /* parte palabras largas */
  font-style:        italic;
}
📐
Unidades CSS más usadas:
px — píxeles, absoluto. rem — relativo al font-size del <html> (recomendado). em — relativo al padre. % — porcentaje del padre. vw / vh — porcentaje del viewport. clamp(min, ideal, max) — valor responsivo automático.

Flexbox
El sistema de layout más versátil para alinear elementos

Flexbox (Flexible Box Layout) es un sistema de diseño unidimensional: organiza elementos en una fila o columna. Es perfecto para alinear elementos, distribuir espacio y crear layouts simples. Se activa con display: flex en el contenedor padre.

flexbox.css FLEXBOX 
/* ===== CONTENEDOR FLEX ===== */
.contenedor {
  display:         flex;
  flex-direction:  row;          /* row | column | row-reverse */
  flex-wrap:       wrap;         /* wrap | nowrap */
  gap:             16px;         /* espacio entre items */

  /* Eje principal (horizontal si row) */
  justify-content: flex-start;   /* inicio */
  justify-content: flex-end;     /* final */
  justify-content: center;       /* centro */
  justify-content: space-between;/* espacio entre */
  justify-content: space-around; /* espacio alrededor */
  justify-content: space-evenly; /* espacio igual */

  /* Eje cruzado (vertical si row) */
  align-items:    stretch;       /* estira a mismo alto */
  align-items:    center;        /* centra verticalmente */
  align-items:    flex-start;    /* arriba */
  align-items:    flex-end;      /* abajo */
}

/* ===== HIJOS FLEX ===== */
.item {
  flex-grow:   1;   /* crece para llenar espacio disponible */
  flex-shrink: 0;   /* no se encoge (0 = no encoge) */
  flex-basis:  200px; /* tamaño base antes de grow/shrink */
  flex:        1 1 200px;  /* shorthand: grow shrink basis */
  align-self:  center;   /* alineación individual */
  order:       2;        /* orden visual (no en DOM) */
}

/* Centrado perfecto (el truco más usado de CSS) */
.centrado {
  display:         flex;
  justify-content: center;
  align-items:     center;
}

CSS Grid
Layout bidimensional: filas y columnas al mismo tiempo

CSS Grid es el sistema de layout más poderoso de CSS. A diferencia de Flexbox (una dimensión), Grid trabaja en dos dimensiones simultáneamente: filas y columnas. Es ideal para estructuras de página completas, galerías y layouts complejos.

css-grid.css GRID 
/* ===== CONTENEDOR GRID ===== */
.galeria {
  display:                grid;
  grid-template-columns:  repeat(3, 1fr);    /* 3 columnas iguales */
  grid-template-columns:  200px 1fr 1fr;    /* mixto */
  grid-template-columns:  repeat(auto-fill, minmax(240px,1fr)); /* responsivo */
  grid-template-rows:     auto 1fr auto;
  gap:                    20px;
  row-gap:                16px;
  column-gap:             24px;
}

/* ===== AREAS DE GRID (layout visual) ===== */
.pagina {
  display:               grid;
  grid-template-areas:
    "header  header  header"
    "sidebar main    main"
    "footer  footer  footer";
  grid-template-columns: 250px 1fr 1fr;
  min-height:            100vh;
}
.header  { grid-area: header;  }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main;    }
.footer  { grid-area: footer;  }

/* ===== POSICIONAMIENTO DE CELDAS ===== */
.destacado {
  grid-column: 1 / 3;     /* ocupa columnas 1 a 3 */
  grid-column: span 2;     /* ocupa 2 columnas */
  grid-row:    span 2;     /* ocupa 2 filas */
}

animaciones.css ANIMACIONES 
/* ===== TRANSITION (cambio suave entre estados) ===== */
.boton {
  background:  #7ab8e0;
  transform:   scale(1);

  /* propiedad | duración | timing | retraso */
  transition: all .3s ease 0s;
  transition: background .2s, transform .2s;
}
.boton:hover {
  background: #3a7bbf;
  transform:  scale(1.06);
}

/* Funciones de timing */
/* ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier() */

/* ===== TRANSFORM (transformaciones visuales) ===== */
.elemento {
  transform: translateX(20px);      /* mover horizontalmente */
  transform: translateY(-10px);     /* mover verticalmente */
  transform: scale(1.1);             /* escalar */
  transform: rotate(45deg);          /* rotar */
  transform: skewX(10deg);           /* inclinar */
  /* Combinar: */
  transform: translateY(-6px) scale(1.02);
}

/* ===== @KEYFRAMES (animación continua) ===== */
@keyframes rebotar {
  0%, 100% { transform: translateY(0);     }
  50%       { transform: translateY(-16px); }
}
@keyframes aparecer {
  from { opacity:0; transform:translateY(20px); }
  to   { opacity:1; transform:translateY(0);    }
}

.pelota {
  /* nombre | duración | timing | delay | iteraciones | dirección */
  animation: rebotar 1.5s ease-in-out 0s infinite;
  animation-fill-mode: both;   /* mantiene el último estado */
}

Diseño Responsivo
Media queries, unidades fluidas y mobile-first

El diseño responsivo hace que una página web se vea bien en cualquier dispositivo: celular, tablet o computadora. La estrategia mobile-first consiste en diseñar primero para pantallas pequeñas y luego añadir estilos para pantallas más grandes con min-width.

responsive.css RESPONSIVE 
/* ===== MEDIA QUERIES ===== */
/* Estrategia mobile-first: base = celular, luego crece */

/* Base: todos los dispositivos (celular) */
.tarjetas {
  display:               grid;
  grid-template-columns: 1fr;     /* 1 columna en móvil */
  gap:                   16px;
  padding:               16px;
}

/* Tablet: min 600px */
@media (min-width: 600px) {
  .tarjetas {
    grid-template-columns: repeat(2, 1fr);  /* 2 columnas */
    padding: 24px;
  }
}

/* Desktop: min 1024px */
@media (min-width: 1024px) {
  .tarjetas {
    grid-template-columns: repeat(3, 1fr);  /* 3 columnas */
    max-width: 1200px;
    margin: 0 auto;
  }
}

/* Media queries especiales */
@media (max-width: 480px) {
  .hero-title { font-size: 1.8rem; }
}
@media (prefers-color-scheme: dark) {
  body { background: #0f172a; color: #e2e8f0; }
}
@media (orientation: landscape) {
  .hero { min-height: 50vh; }
}

/* Grid responsivo sin media queries (la magia) */
.auto-grid {
  display:               grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 24px;
}
📱
min-width
Aplica estilos cuando la pantalla es igual o más grande. Núcleo del mobile-first.
🖥️
max-width
Aplica estilos cuando la pantalla es igual o más pequeña. Útil para ajustes específicos.
🎛️
clamp()
Valor responsivo en una sola línea: mínimo, ideal y máximo. Perfecto para font-size.
🌗
prefers-color-scheme
Detecta si el usuario prefiere modo oscuro o claro en su sistema operativo.
🔄
auto-fill / auto-fit
En CSS Grid, crean columnas automáticamente según el espacio disponible. Sin media queries.
📐
aspect-ratio
Mantiene una proporción fija (ej: 16/9 para video). Clave para elementos responsivos.
